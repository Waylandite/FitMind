## 6. 消息模板

### 6.1 完整状态记录

```
"昨晚{睡眠}，今天{疲劳}，{压力}，{部位}{酸痛}，{体重}，精神状态{情绪}"
```

### 6.2 简化状态

```
"今天{疲劳}，{部位}{酸痛}"
"状态{一般/还行}"
"{睡眠}，{疲劳}"
```

### 6.3 只有体重

```
"体重{kg}"
"今天体重{kg}kg"
```

### 6.4 极简描述

```
"状态一般"
"感觉还行"
"今天有点累"
"恢复得不错"
```

## 8. 组合示例

### 数值 + 定性混合

```
输入: "昨晚睡得很差大概5小时，今天累到不行，下背酸了三天还在酸，压力还好"
期望:
  sleep_hours: 5
  fatigue_level: 9
  stress_level: 4
  soreness_level: 7
  soreness_location: 下背
```

### 只有定性无数字

```
输入: "今天状态很好，完全恢复了，精力充沛"
期望:
  mood: 很好
  fatigue_level: 2
```
